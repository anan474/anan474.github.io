title: Membuat Catatan Ini
---

Blog ini dibuat berdasarkan template dari [grundgesetz-skeleton](https://github.com/kata-ai/grundgesetz-skeleton) yaitu template untuk membuat dokumentasi atau dalam kasus saya yaitu blog pribadi, template ini di susun oleh tim dari [kata.ai](https://github.com/kata-ai).

Situs ini (dari template tersebut) dibangun dengan basis _framework_ Gatsby, yaitu kerangka kerja React.js untuk membangun web yang di _render_ dari server (_server side rendering_).

Situs ini di _hosting_ di netlify.

Ingin membuat serupa situs ini? cek [dokumentasi grundgesetz](https://grundgesetz-skeleton.now.sh/).
